# Deep Research

Use this skill when the task needs multiple rounds of planning, searching, evaluation, and synthesis.

## Workflow

1. Start a background research task with the `research` tool when the question is broad, uncertain, or evidence-heavy.
2. Prefer iterative coverage over one-shot answers.
3. Add context with the `research` tool or `/context` when the user clarifies constraints mid-run.
4. Use milestone summaries and the final synthesis as factual context for later turns.

## Convergence

- Stop when confidence is high and no critical gaps remain.
- Keep unresolved uncertainty explicit in the final answer.
- Prefer citing concrete sources over broad unsupported claims.
